# 🐾 Pet Love

O **Pet Love** é um projeto de extensão desenvolvido por Marcus Vinicius, aluno de Engenharia de Software no **iCEV**. 

O objetivo do aplicativo é funcionar como uma rede de conexões para pets, permitindo que tutores encontrem outros animais para socialização, passeios ou cruzamento, facilitando a vida de quem busca o par ideal para o seu melhor amigo.

## 🚀 Tecnologias Utilizadas
* **Frontend:** React Native (TypeScript)
* **Backend:** Supabase (PostgreSQL)
* **Ferramentas:** Android Studio, Git/GitHub
